# Sarroff1

Sarroff1 is a full-stack web application built using React and Node.js.  
It demonstrates how frontend and backend systems can work together via REST APIs to manage tasks efficiently.  
The project is designed to showcase practical development skills including API integration, CRUD operations, and a clean, responsive UI.

---

## 🚀 Features

- Full CRUD functionality for tasks
- REST API integration
- Frontend & Backend connection
- Asynchronous requests using Axios / Fetch
- Responsive user interface
- Modular code structure (easy to scale)

---

## 🛠 Tech Stack

**Frontend:**  
- React  
- JavaScript (ES6+)  
- CSS / Tailwind  

**Backend:**  
- Node.js  
- Express.js  

**Database:**  
- MongoDB
